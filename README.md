# PIBICEM
Esse repósitorio servirá para postar os materiais produzidos durante a execução de um projeto de Iniciação Cientifíca do Ensino Médio no Departamento de Matemática da UFU (IME-UFU) intitulado de Uma Abordagem Teórica e Histórica das Séries Infinitas
<br>
Nesse repositório serão postados:
<br>
-Vídeos produzidos com a linguagem Manim em Python
<br>
-Uma calculadora de séries de Taylor no terminal em Python 
<br>
-Um modelo de maching learning baseado em séries de Taylor para análise de EEGs
